The "page" package
==================
The page package provides base classes and utilities to manipulate
webpages using the selenium driver.

The "page" module
-----------------
:mod:`selenium_page.page` contains base classes representing abstract webpages using
:class:`~page.element.BasePageElement` data descriptor attributes. These class attributes
represent various elements on the page providing an easier API for manipulating the DOM.
